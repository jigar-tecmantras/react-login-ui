# React Login UI

A standalone React login page with polished, glassmorphic styling. The project focuses on UX/UI best practices, providing a single-screen experience that could later be wired to a backend if needed.

## Setup

```bash
cd frontend
npm install
npm start
```

## Features

- Fully client-side form validation for email and password inputs.
- Disabled submit button until required fields are populated.
- Responsive layout with gradient background, glow effects, and branded card.
- Helpful links for password recovery and account creation, plus social sign-in buttons for visual completeness.

## Notes

- This repository contains frontend-only code (no backend/API integrations).
- Using Create React App conventions so the experience is ready for typical React workflows.
